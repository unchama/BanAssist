# BanAssist
config.ymlで指定したjsonファイルからcompromised accountのみを抽出し、アクセスを禁止するプラグイン。<br>
初期設定ではJPMCPvPが公開している処罰リストを(勝手に)お借りしています。<br>
誤検知も考慮して、/ignoreコマンドでプレイヤーを追加すると除外処理されます。<br>
コードを使用される場合は自己責任でご利用下さい。<br>
